==========
	DE
==========
Um die List & Label FireDAC VCL-Komponente in Ihrem C++ Builder Projekt verwenden zu können, müssen Sie das in Delphi bereitgestellte Package verwenden.
Das VCL Package liegt im "<List & Label Installationspfad>\Beispiele\Delphi\FireDAC\Component\" Verzeichnis, um das Package zu verwenden gehen Sie bitte wie folgt vor:
- Öffnen Sie RAD Studio.
- Wählen Sie im IDE-Menü 'Komponente > Packages installieren', im Listenfeld 'Entwurfszeit-Packages' wird eine Liste der verfügbaren Packages angezeigt. Wählen Sie dort die "combit List & Label ?? Components" aus und entfernen Sie diese.
- Beenden Sie RAD Studio.
- Verwenden Sie MSbuild um das "combit List & Label ?? FireDAC VCL"-Package zu erzeugen:
  - Zum Aufrufen von MSBuild in einer benutzerdefinierten Befehlsumgebung wählen Sie 'Start | Programme | Embarcadero RAD Studio <Version> | RAD Studio-Befehlszeile'.
  - Führen Sie folgenden Befehl in der RAD Studio-Befehlszeile aus: msbuild "<List & Label Installationspfad>\Beispiele\Delphi\FireDAC\Component\ListLabel??Components.dproj"
- Wenn Sie das Package erzeugt haben, wird die gemeinsam genutzte Package-Bibliotheksdatei standardmäßig in folgendem Verzeichnis generiert: "C:\Users\Public\Documents\Embarcadero\Studio\<Version>\Bpl"
- Öffnen Sie RAD Studio erneut.
- Wählen Sie im IDE-Menü 'Komponente > Packages installieren' aus und fügen Sie das "ListLabel??Components.bpl" Package aus dem oben beschriebenen Verzeichnis hinzu. Nun sollte die Komponente in der Palette zu sehen sein.

==========
	EN
==========
To use the List & Label FireDAC VCL component in your C++ Builder project you have to use the package created in Delphi.
The VCL package is located in the "<List & Label installation path>\Samples\Delphi\FireDAC\Component\" directory, to use the package please proceed as follows:
- Open RAD Studio.
- Select 'Component > Install Packages' in the IDE menu, a list of available packages is displayed in the 'Design Time Packages' list box from there, select the "combit List & Label ?? Components" and remove it.
- Exit RAD Studio.
- Use MSbuild to generate/build the "combit List & Label ?? FireDAC VCL" package:
  - To invoke MSBuild in a custom command environment, select 'Start | Programs | Embarcadero RAD Studio <Version> | RAD Studio Command Line'.
  - Run the following command on the RAD Studio command line: msbuild "<List & Label installation path>\Samples\Delphi\FireDAC\Component\ListLabel??Components.dproj".
- When you have created the package, the shared package library file is generated by default in the following directory: "C:\Users\Public\Documents\Embarcadero\Studio\<Version>\Bpl".
- Open RAD Studio again.
- From the IDE menu, select 'Component > Install Packages' and select the "ListLabel??Components.bpl" package from the directory described above. Now you should see the component in the palette.
