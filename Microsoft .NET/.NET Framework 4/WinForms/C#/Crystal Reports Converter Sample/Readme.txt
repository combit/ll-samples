==========
	DE
==========
Um das Crystal Reports Converter-Beispiel zu verwenden, benötigen Sie die folgenden Assemblies:
- CrystalDecisions.CrystalReports.Engine
- CrystalDecisions.Shared

Bei Verwendung von Crystal Reports Version 2020 oder höher ist mindestens die Assembly-Version 13.0.31 erforderlich. Für ältere Versionen können die Assemblies aus dem NuGet-Package "CrystalDecisions.CrystalReports.Engine" (Version 1.0.0) verwendet werden.

Die Visual Studio Runtimes (SDK) für Crystal Reports können Sie unter dem folgenden Link herunterladen:
https://wiki.scn.sap.com/wiki/display/BOBJ/Crystal+Reports%2C+Developer+for+Visual+Studio+Downloads

==========
	EN
==========
To use the Crystal Reports Converter Sample you need the following assemblies:
- CrystalDecisions.CrystalReports.Engine
- CrystalDecisions.Shared

When using Crystal Reports version 2020 or higher, at least assembly version 13.0.31 is required. For older versions, the assemblies from the NuGet package "CrystalDecisions.CrystalReports.Engine" (version 1.0.0) can be used.

The Visual Studio Runtimes (SDK) for Crystal Reports can be downloaded under the following link:
https://wiki.scn.sap.com/wiki/display/BOBJ/Crystal+Reports%2C+Developer+for+Visual+Studio+Downloads
